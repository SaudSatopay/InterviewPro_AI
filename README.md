InterviewPro AI is a full-stack mock interview preparation platform. Upload a job description and resume, then practice with AI-generated questions tailored to your target role. Answer by voice (Web Speech API) or code, get scored in real-time by Groq/Llama 3.3, and receive a personalized study roadmap with weekly breakdowns. Built with React, TypeScript, Vite, Tailwind CSS, Framer Motion, and Express.

Key features:

Voice-based answering with live audio waveform visualizer
Question navigation with skip and 2-min timer per question
AI-powered preparation roadmap generator with weekly plans
Interview history tracking with session playback
Dark futuristic UI with smooth Framer Motion animations
